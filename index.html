ⓘ установка

☁ Android

  1. Скачай APK на планшет/телефон
  2. Открой файл — Android попросит разрешить установку
  3. Разреши Установка из неизвестных источников
  4. Нажми Установить

⌨ Windows

  1. Скачай Termify.exe
  2. Запусти — браузер готов к работе
  3. Требуется WebView2 Runtime (обычно уже есть в Windows 11)
  4. При первом запуске может быть задержка — распаковывается .NET
