ⓘ установка

☁ Android

  1. Скачай APK на планшет/телефон
  2. Открой файл — Android попросит разрешить установку
  3. Разреши Установка из неизвестных источников
  4. Нажми Установить

⌨ Windows

  1. Скачай Termify.exe
  2. Установи .NET 8 Runtime если ещё нет
  3. Запусти — браузер готов к работе
  4. Требуется WebView2 Runtime (обычно уже есть в Windows 11)
